MODIS_Landsat_Sentinel2星源分幅简述【20250715】

发表于2025-07-15|更新于2025-07-15|遥感数据
|字数总计:1.4k|阅读时长:4分钟|阅读量:

遥感影像分幅是指将覆盖较大地理区域的连续遥感影像数据，按照一定的规则或标准切割成多个较小、更易管理的矩形(或规则形状)图幅的过程。这是遥感数据处理、存储、分发和应用中一项非常基础和关键的操作。遥感影像分幅的必要性：

  1. 管理便利性：
    • 大范围(如全省、全国、全球)的遥感影像数据量极其庞大(可达TB甚至PB级)，直接处理整景数据非常困难，效率低下。
    • 分幅后，每个图幅数据量较小，便于存储、备份、检索、传输和更新。
  2. 处理效率：
    • 许多影像处理操作(如辐射定标、大气校正、几何校正、信息提取、深度学习模型训练/推理)在较小图幅上进行计算效率更高，资源消耗更少。
    • 便于分布式计算和并行处理。
  3. 标准化与共享：
    • 与标准地形图分幅系统(如1:100万、1:25万、1:5万等)保持一致，便于与矢量地图数据、地形图以及其他来源的栅格数据进行叠加分析、融合和集成。
    • 方便数据交换、分发和按区域订购。
  4. 应用需求：
    • 用户往往只需要特定区域(如某个县、市、项目区)的数据，分幅允许按需提供所需图幅，避免下载或处理冗余数据。
    • 制作标准比例尺地图产品的基础。

⛄MODIS

  • 中分辨率成像光谱仪

  • 卫星平台：Terra(1999)、Aqua(2002)

  • 运营机构：NASA

（1）核心特性

参数 说明
幅宽 2330km(全球无缝覆盖，单日成像范围极广)
数据类型 36波段多光谱数据(0.4-14.4μm)，覆盖可见光至热红外
存储格式 HDF格式
空间分辨率 250m(波段1-2)
500m(波段3-7)
1km(其余波段)
时间分辨率 每日全球覆盖(Terra上午过境+Aqua下午过境)
分幅系统 正弦投影网格(Sinusoidal Grid)
分幅标准：36列(h00-h35)×18行(v00-v17)
格网大小：约1200×1200km→144万km²(赤道)
图幅编号：hXXvYY(如h08v05)
处理等级 L1B：辐射定标辐亮度
L2/L3：地球物理参数(植被指数/地表温度)
L4：模型同化产品(如温度同化数据集)
典型产品 MOD09GA(地表反射率)、MOD11A1(地表温度)、MOD13Q1(植被指数)

（2）分幅系统

（3）优势与局限

  • 优势：全球每日监测能力，20年+历史数据，适合大尺度环境研究(如气候变化)。
  • 局限：分辨率低(≥250m)，城市级精细分析不适用。

（4）典型应用

  • 全球植被动态监测(NDVI/EVI)
  • 火灾热点探测(热红外波段)
  • 海洋叶绿素浓度反演

⛄Landsat

  • 陆地卫星系列

  • 在役卫星：Landsat-8(2013)、Landsat-9(2021)

  • 运营机构：NASA/USGS

（1）核心特性

参数 说明
幅宽 185km(推扫式成像，单景覆盖适中)
数据类型 11波段多光谱数据(0.43-12.51μm)，覆盖可见光至热红外
存储格式 TIFF格式
空间分辨率 全色：15m
多光谱：30m
热红外：100m(产品中重采样至30m)
时间分辨率 16天(单星)→8天(Landsat-8/9双星联合)
分幅系统 全球参考系统(WRS-2)
分幅标准：轨道号(Path：001-233)和行号(Row：001-248)
格网大小：185(东西)×180(南北)km→3.33万km²
图幅编号：Path/Row(如123032)
处理等级 L1TP：地形校正正射影像(几何精度≤12m)
L2SP：地表反射率+地表温度(大气校正)
典型数据 Tier 1(最高质量，几何精度<12m)

（2）分幅系统

（3）优势与局限

  • 优势：历史最长(1972至今)，辐射定标精准，支持长时间序列生态研究。
  • 局限：重访周期较长(8天)，易受云层干扰。

（4）典型应用

  • 土地利用/土地覆盖分类
  • 城市热岛效应分析(热红外波段)
  • 冰川退缩监测

⛄Sentinel2

  • 哨兵2号

  • 卫星平台：Sentinel-2A(2015)、Sentinel-2B(2017)、Sentinel-2C(2024，接替2A)

  • 运营机构：欧洲航天局(ESA)

（1）核心特性

参数 说明
幅宽 290km(双星协同覆盖能力极强)
数据类型 13波段多光谱数据(0.443-2.280μm)，覆盖可见光至短波红外
存储格式 SAFE格式
空间分辨率 10m(4个，可见光/近红外波段)
20m(6个，红边/短波红外波段)
60m(3个，大气校正波段)
时间分辨率 10天(单星)→5天(双星全球覆盖)
分幅系统 军事网格参考系统(MGRS)
分幅标准：按100km×100km UTM网格分幅
格网大小：100km×100km(UTM平面坐标)→1万km²
图幅编号：UTM带号+纬度带+网格编码(如32TMT)
处理等级 L1C：正射校正表观反射率
L2A：大气校正地表反射率(Sen2Cor算法生成)
特色波段 红边波段(704nm/740nm/783nm)、海岸气溶胶波段(443nm)

（2）分幅系统

（3）优势与局限

  • 优势：高分辨率(10m)+高频次(5天)，红边波段支持农业精细化监测。
  • 局限：无热红外波段，无法直接获取地表温度。

（4）典型应用

  • 农作物长势监测与产量预估(红边波段)
  • 森林病虫害检测
  • 海岸带环境调查(海岸气溶胶波段穿透性强)
